# 开窍 🧠

> **装了开窍的Agent，会记住你、会自己变聪明、会在你觉得之前先想到——然后像朋友一样问你一句。**

## 开窍是什么

开窍 = Agent从"被动工具"变成"主动搭档"的那一刻。

竞品全是被动的——你问它才答，你让它记才记。开窍的杀手锏是**主动**：
- 不会等到犯错才记录，而是提前预警"上次这样做不好"
- 不会等到主人来问，而是过段时间主动说"我们对这件事的理解还一致吗？"
- 不会用AI味的话确认，而是像人一样问"你是不是想…"

## 五大核心能力

| 能力 | 一句话说明 |
|------|-----------|
| **主人画像** | 不是填表，是理解——从对话中逐渐学习这个人 |
| **经验三档置信度** | 假设→经验→铁律，不是一视同仁 |
| **场景化经验存储** | 按"你在做什么+遇到了什么"组织，不是按分类 |
| **主动询问（去AI化）** | 当下问/事后问/过段时间问，说人话不说客服话 |
| **认知漂移对齐** | 定期检查"我们之前说好的那件事，还适用吗？" |

## 最值钱的功能：过段时间问

这是竞品完全没有的功能。

人和Agent的认知都会漂移——主人可能改了偏好但没说，Agent可能把临时习惯当成了铁律。开窍每隔一段时间主动问一句，防止双方渐行渐远。

## 核心闭环

```
记住 → 发现缺口 → 问 → 记住更多 → 进化 → 预警过时 → 确认 → 更新
```

## 与已有系统的关系

开窍**不替代**MEMORY.md / SOUL.md / TOOLS.md，而是**增强**它们。核心原则：不造新轮子，用好已有的。开窍是粘合剂，不是替代品。
